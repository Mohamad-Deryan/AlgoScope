# AlgoScope

## Interactive Algorithm Learning Platform

AlgoScope is a full-stack web application designed to help students understand algorithms through interactive visualization.

## Features

- Step-by-step algorithm execution
- Sorting visualization (Selection Sort)
- Recursion visualization (Factorial)
- Graph algorithms (BFS, Dijkstra)
- Pseudocode highlighting
- Explanation panel for each step
- Interactive controls (Play, Next, Previous, Reset)
- Quiz system for learning reinforcement

## Technologies Used

### Frontend
- React
- JavaScript
- HTML/CSS

### Backend
- Spring Boot
- Java

### Communication
- REST APIs
- JSON data exchange

## How It Works

1. User selects an algorithm
2. Frontend sends request to backend
3. Backend processes algorithm and generates steps
4. Steps are returned as JSON
5. Frontend visualizes execution step-by-step
